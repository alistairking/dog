# dog
dog > cat

## Installing
```
pip3 install .
```

## Using

Use your dog just like you'd use your cat. It just won't complain as much.

```
$ dog s3://my-s3-bucket/cool-file.gz ~/tmp.txt | wc -l
   12224
```
